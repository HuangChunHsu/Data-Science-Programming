getdoc <- function(url)
{
html <- htmlParse( getURL(url) )
doc <- xpathSApply( html, “//div[@id='main-content']”, xmlValue )
time <- xpathSApply( html, "//*[@id='main-content']/div[4]/span[2]“, xmlValue )
temp <- gsub(” “,” 0“, unlist(time) )
part <- strsplit( temp, split=” “, fixed=T )
#date <- paste(part[[1]][2], part[[1]][3], part[[1]][5], sep=”-“)

Conclusion :

1.從時間與前五關鍵字的圖可以發現，有四葉大量出現的時候總是伴隨著一花，因此可以猜測兩者十分有關，而事實上兩者都是“五等分的花嫁”中的人物，而春場雖然出現次數不如一花與四葉高，但是峰值出現時間相同，因此判斷春場也與五等分的花嫁有關，而事實上，春場正是五等分的花嫁的作者
2.文章的數量以17、19以及22、00有極值，猜測17是養生工程師的通勤文，19則是吃完飯後繼續發，22則是輪班工程師的通勤文，00是睡前晚安文
3.從文字的群集來看可以發現，一花、四葉、五月都是五等分的花嫁的出場人物，因此自成一國。伊井野、石上、學姊都是“輝夜大小姐想讓我告白～天才們的戀愛頭腦戰～”的角色。六花和花王雖然很近，但事實上是jieba斷詞的失誤，六花與花王應該是六花王被截斷的結果，事實上是因希洽版名人“Makinoyui”有著六花王的稱號。大腦、痛覺、模擬則能推測出鄉民們應該是在討論VR遊戲。小學生和鬥智則能推出應該是在討論“龍王的工作”，因為其中有大量小學生下將棋的內容，抄襲、本傳、骨架、質量應為討論繪師抄襲的議題，阿比和生日快樂十分接近，故猜測7/12為阿比蓋爾的生日，經查證Abigail Williams確實生於1680年的7/12。
